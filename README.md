✨ FeaturesUser Authentication: 
Secure signup login, and logout via Firebase.
Product Catalog: Dynamic display of sauces and spices with filtering.
Shopping Cart: Real-time cart updates, item counters, and price calculations.
Responsive Design: Fully optimized for mobile, tablet, and desktop screens.
Admin Dashboard: Interface to add, update, or remove inventory items.

🛠️ Tech Stack 
Frontend: HTML5, CSS3, JavaScript (ES6+) 
Backend & Database: Firebase Firestore 
Authentication: Firebase Auth 
Hosting: Firebase Hosting 
Development Environment: Firebase Studio

Configure Firebase 
Create a project in the Firebase Console.
Enable Firestore Database and Authentication (Email/Password).
Copy your Firebase configuration object.
Replace the placeholder config in src/firebase.js with your keys.
